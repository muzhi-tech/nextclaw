# NextClaw 产品需求文档 (PRD)

> V1 版本 - OpenClaw 人设配置优化 Skill

## 1. 产品概述

### 1.1 产品定位

NextClaw 是一个 OpenClaw Skill，帮助用户快速创建和优化 AI 助手的人设配置（SOUL.md、USER.md 等）。

### 1.2 目标用户

| 用户类型 | 特征 | 核心需求 |
|---------|------|---------|
| 新手用户 | 刚接触 OpenClaw | 快速生成可用的人设 |
| 进阶用户 | 已有基础配置 | 精细调整和优化 |
| 开发者/极客 | 深度定制需求 | 模板复用和扩展 |

### 1.3 版本规划

- **V1 (当前)**: Skills 方式，发布到 GitHub，面向个人用户
- **V2 (未来)**: 独立平台，支持团队协作

---

## 2. 核心功能

### 2.1 功能列表

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 创建新人设 | 通过对话式交互创建 SOUL.md | P0 |
| 优化现有人设 | 分析并优化已有配置 | P0 |
| 模板库 | 内置 10+ 常用人设模板 | P0 |
| 示例对话预览 | 部署前预览人设效果 | P1 |
| 自动备份 | 写入前备份原文件 | P1 |
| 一键部署 | 直接写入 OpenClaw 工作目录 | P1 |

### 2.2 用户流程

```
用户输入 → 意图识别 → 执行操作 → 生成结果 → 用户确认 → 写入文件
```

---

## 3. 技术设计

### 3.1 项目结构

```
nextclaw/
├── src/
│   ├── index.ts        # 入口
│   ├── creator.ts      # 创建器
│   ├── optimizer.ts    # 优化器
│   └── templates.ts    # 模板管理
├── templates/          # 人设模板
├── docs/               # 文档
└── tests/              # 测试
```

### 3.2 核心模块

- **SoulCreator**: 人设创建逻辑
- **SoulOptimizer**: 人设优化逻辑
- **TemplateManager**: 模板管理

## 4. 异常处理

### 4.1 异常分类

| 类型 | 示例 | 处理策略 |
|------|------|----------|
| 文件操作 | 文件不存在 | 提示创建新文件 |
| 用户输入 | 信息不完整 | 追问缺失项 |
| AI 生成 | 生成超时 | 降级使用模板 |
